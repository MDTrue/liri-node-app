# liri-node-app
========================================================================
## Language Interpretation and Recognition Interface;
This is a call and response node.js command line interface that makes use 
of several API's and node.js packages including:

* inquirer
* dotenv
* moment
* node-spotify-api
* OMDB-api
* bandsintown-api
